# Customer Analytics Dashboard | Power BI

## Overview

This project is an interactive **Power BI dashboard** designed to analyze customer data and provide actionable business insights. The dashboard enables users to explore customer demographics, segmentation, and detailed customer information through interactive visualizations.

## Dashboard Pages

### 1. Customer Demographics
Provides an overview of customer characteristics, including:
- Customer distribution
- Age analysis
- Gender breakdown
- Key KPI cards
- Interactive donut charts

### 2. Customer Segmentation
Analyzes customer groups using:
- Bar charts
- Segment comparisons
- Customer category analysis
- Interactive filtering

### 3. Customer Details
Displays detailed customer-level information with:
- Interactive data table
- Page navigation
- Drill-down capabilities

## Features

- Interactive slicers and filters
- Modern dashboard design

- Multi-page navigation
